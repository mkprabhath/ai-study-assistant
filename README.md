# AI Study Assistant

A browser-based study assistant that converts raw study material into structured
summaries, bullet points, and exam-ready notes using JavaScript logic inspired by
generative AI workflows.

## Features
- Text summarization
- Bullet point generation
- Exam-oriented note formatting
- Runs entirely in the browser

## Tech Stack
- HTML
- JavaScript

## Status
Core functionality complete. Future scope includes integration with real
Generative AI APIs.
